Operadores

Aritméticos:

  • Adição
    5+2==7
  • Subtração
    5-2==3
  • Multiplicação
    5*2==10
  • Divisão
    5/2==2.5
  • Expoenciação
    5**2==25
  • Resto
    5%2==1

    Precedência:

    1. ( )
    1. **
    1. * , / , %
    1. + e -

    Atribuição:

    Simplificação:

    n=n+10 → n+=10

    Soma/ Subtração 1:

    x=x+1 → x++

Relacionais:

  • Maior:
    5>2 → true
  • Menor:
    7<4 → false
  • Maior ou igual:
    8>=8 → true
  • Menor ou igual:
    9<=7 → false
  • Igual:
    5==5 → true
  • Diferente:
    4!=4 → false

    Identidade:

    5==5    → true
5=='5'  → true

5==='5' → false
5===5   → true

Lógicos:

  • ! (Operador 'no'):
    !true  → false
!false → true
  • && (Operador 'and'):
    true  && true  → true
true  && false → false
false && true  → false
false && false → false
  • || [pipe] (operador 'or'):
    true  || true  → true
true  || false → true
false || true  → true
false || false → false

Ternário:

true or false?  true  :  false
idade>= 18 ?    maior :  menor
